PHP 陣列操作的 10 個實用技巧

更新於 2024-12-23 分類於 技術筆記 閱讀次數：

整理了些 PHP開發 在 工作或解題上可能會用到的實用函數，

依需求可大致區分為以下類別，

是否存在特定資料

二維陣列 使用 array_key_exists()

一維陣列 使用 in_array()

提取特定欄位資料

使用 array_column()

索引跟資料翻轉顯示

使用 array_flip()

計算資料存在數量

使用 array_count_values()

填充多個預設值

使用 array_fill()

特定位置移除/拼接數值

名稱上容易混淆的就放一起講

**array_splice(拼接)**，移除陣列特定位置數值 並用 其他數值 取代

**array_slice(移除)**，切除陣列特定位置數值

物件資料轉陣列

使用 json_decode()

合併多個陣列資料

只有兩個，使用 array_combine()

多個陣列，使用 array_merge()

取得索引或資料的數值

要取得 資料，使用 array_values()

要取得 索引，使用 array_keys()

處理多次資料功能循環

多個陣列，使用 array_map()

陣列相關的函數還有很多，這邊筆者只是先列出較常使用的提供參考。

後端通常情況下回傳資料都會是 json 格式，前端轉陣列使用上的參考。
