Markdown語法筆記

發表於2021-04-07更新於2021-08-16

作個紀錄

標題

#當標題使用，越少字越大，最多6個

## 第二段 (H2)
### 第三段(H3)

效果如下

第二段 (H2)

第三段(H3)

刪除線

句子前後都加 ~~ 會有刪除線效果

~~我是刪除線~~

效果如下

我是刪除線

清單

前面一個 * 當一般清單(就是沒有1.、2.、3.、…etc)

* 第一個
 * 第二個
* 第三個

效果如下

  • 第一個
  • 第二個
  • 第三個

文字大區塊

要用大區塊將文字包起來可以按四個空白或是一個Tab

echo "Hello World!"

文字小區塊

要用小區塊將文字包起來前後都加上 ` 反引號

echo "Hello World!"

程式碼區塊

程式碼區塊在開頭結尾加上```可以顯示

要加上Highlight 以JS為例就是 js 或 javascript 在前面的 ``` 加上 js 或 javascript 就有囉

超連結

需要超連結的格式

[要顯示的名稱](網址)

圖片

![若圖片顯示不出來的文字](圖片連結位置或網址)

效果如下，用原生的沒辦法設定大小，可以改用HTML的　img　標籤來設定
